# Bezier — Pre-launch Video

Bezier の pre-launch（waitlist）動画プロジェクト。LP と同じ世界観・同じ言葉で、waitlist 登録を獲得する 60–75s。

## 確定事項（2026-06-25）
- **尺/配信**: LPヒーロー埋め込み **60–75s**
- **制作**: **ハイブリッド**（実画面録画 ＋ Remotion/モーション）
- **言語**: **英日両対応**（英語マスター＋日本語）
- **コンセプト**: **A2「The Orchestrator, for designers」**。Bezier = Superset/cmux のような AI Agent Orchestrator を**デザイナー向けに再設計**したもの。エンジニア向け orchestrator の売り方を踏襲し、**「diff/PR→動く画面」「読む/merge→クリック/注釈」の2語差し替え**でデザイナー化。
- **背骨**: 自己像シフト — HERO「コードは書かない。プロダクトを指揮する。」／ CLOSE「つくる人から、導く人へ。」**名前/ロゴ由来の比喩・署名グラフィックは使わない**（ビジュアル未確定のため）
